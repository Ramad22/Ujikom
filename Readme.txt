Untuk instalasi website badmintonnekat ikuti petunjuk di bawah ini :

1. Install xampp.
2. Setelah xampp terinstall, silahkan exstract Ujikom_gilang&malik.zip, kemudian pindahkan file hasil exstract tadi ke folder htdocs -> c:/xampp/htdocs.
3. Kemudian buatlah sebuah database di phpmyadmin.
4. Upload / import file korban.sql ke database yang baru dibuat.
5. Akses aplikasi di localhost -> localhost/Ujikom_gilang&malik.
6. Selesai
